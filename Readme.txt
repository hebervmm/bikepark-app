BikePark App
O BikePark é um aplicativo React Native que permite aos usuários encontrar facilmente estações de bicicletas próximas do Itaú utilizando a API de dados abertos da Prefeitura do Recife.

Funcionalidades
Busca de estações de bicicletas próximas com base na localização atual do usuário.
Exibição de marcadores no mapa para as estações encontradas.
Informações detalhadas sobre cada estação ao clicar nos marcadores, incluindo capacidade e localização.
Pré-requisitos
Para executar este aplicativo localmente, você precisa ter instalado:

Node.js
Expo CLI
Emulador Android/iOS ou dispositivo físico com Expo Go instalado

Como Usar
Ao abrir o aplicativo, pressione o botão "Buscar Bicicletários Próximos" para permitir o acesso à sua localização.
Aguarde enquanto o aplicativo busca as estações de bicicleta próximas.
Os marcadores no mapa indicam as localizações das estações encontradas.
Clique nos marcadores para visualizar mais informações sobre cada estação, incluindo capacidade e localização.
